# AirWatch - IoT

Módulo de Internet das Coisas do projeto AirWatch.

## Status

**Em desenvolvimento** – os códigos e documentação serão adicionados conforme a evolução do projeto.

## Escopo

Este repositório conterá o firmware para ESP32, integrando sensores de temperatura, umidade e qualidade do ar, com comunicação Wi-Fi, MQTT, endpoints JSON e interface local.

## Como usar (em breve)

Assim que o código estiver disponível, as instruções de execução (Wokwi ou hardware real) serão adicionadas aqui.
